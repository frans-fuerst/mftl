Installation / usage
--------------------

Install dependencies:

    pip3 install --user ujson PyQt4 PythonQwt

Get the repo:

    git clone https://github.com/frans-fuerst/mftl

Fetch some trade history, eg.

    cd mftl
    ./mftl-cli.py fetch BTC_ETH 70000

Display trade curves:

    ./mftl-cli.py show
